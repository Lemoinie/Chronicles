# Chronicles

A modular, high-performance MMORPG framework for Minecraft servers.

## 🚀 Overview
Chronicles is designed to be a robust foundation for RPG servers, focusing on modularity, scalability, and ease of use. It provides a comprehensive suite of modules for player progression, stats, abilities, and more.

## ✨ Key Features
- **Strictly Modular**: Features are isolated and easy to maintain or replace.
- **Event-Driven**: Clean communication between modules using a central event bus.
- **Performance Focused**: Asynchronous operations and optimized data structures.
- **Extensible API**: Easy to hook into and extend for custom features.

## 📦 Modules
- **Profile & Stats**: Core player data and attribute system.
- **Origin & Class**: Unique player backgrounds and RPG classes.
- **Skills & Jobs**: Progression systems for combat and utility.
- **Abilities**: Powerful active and passive skills with cooldown management.
- **Quests**: Integrated questing engine.

## 🛠️ Getting Started
### Building from Source
1. Clone the repository.
2. Build using the provided Gradle wrapper:
   ```bash
   ./gradlew build
   ```
3. Find your jar in `build/libs/`.
